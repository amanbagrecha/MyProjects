# How to use the organiser.
#### Useful for Lecturers, Data entry Folks, anyone who uses in their daily workflow
> **This macro enables you to**
> - Add Features
> - Add new record
> - Delete record
> - Delete feature (except name)
> - Search and query

***It validates the input as well***
